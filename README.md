# LocalAI - Automated Google Business Profile Optimization Platform

A comprehensive multi-tenant SaaS platform for automating Google Business Profile optimization using AI.

## Features

### Core Functionality
- AI-Powered Content Generation - Gemini Pro integration for descriptions, posts, and review replies
- GMB Profile Management - Connect and optimize Google Business Profiles
- Post Scheduling - Schedule and auto-publish GMB posts
- Review Management - AI-generated review responses with sentiment analysis
- Competitor Intelligence - Track and analyze competitors
- Geo-Rank Tracking - Monitor local search rankings with heatmap visualization
- QR Code Generation - Create QR codes for review collection
- Blog Automation - Auto-publish to Blogger with AI-generated content
- Image Generation - AI-powered image creation

### Subscription & Billing
- Multi-payment gateway support (Razorpay, Stripe, PayPal)
- Flexible plan management (Starter, Pro, Agency, Enterprise)
- 7-day free trials, coupon system, invoice management

### Tech Stack
- Frontend: React 18 + TypeScript + Vite + Tailwind CSS
- Backend: Supabase (PostgreSQL + Auth + Edge Functions)
- AI: Google Gemini Pro
- APIs: Google Business Profile, Google Places, Blogger
